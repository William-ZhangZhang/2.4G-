
# STM32F103ZET6 小车主控 接收端

-------------------------------------------------------------------------------------------
	
硬件资源:
	1,指示灯：LED0-(PF8), LED1-(PF9)
	2,串口1打印输出：(波特率:115200,PA9/PA10)
	3,NRF24L01模块：
			CE-(PA1)
			CSN-(PA0)
			IRQ-(PA2)
			MOSI-(PA7)
			MISO-(PA6)
			SCK-(PA5)
	4,无刷电机驱动：
			POWER-(PE10)
			PWM_1-(PB10)
			PWM_2-(PB11)
			BK_LEFT-(PE12)
			BK_RIGHT-(PF0)
			FB_LEFT-(PE14)
			FB_RIGHT-(PF2)
	5,超声波模块：
			Echo-（PB0）
			Trig-（PB1）
