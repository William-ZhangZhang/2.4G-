
# STM32F103ZET6 遥控手柄 发送端

-------------------------------------------------------------------------------------------
	
硬件资源:
	1,指示灯：LED0-(PF8), LED1-(PF9)
	2,串口1打印输出：(波特率:115200,PA9/PA10)
	3,NRF24L01模块：
			CE-(PA1)
			CSN-(PA0)
			IRQ-(PA2)
			MOSI-(PA7)
			MISO-(PA6)
			SCK-(PA5)
	4,摇杆：（低电平触发）
			UP-(PF0)
			DOWN-(PF2)
			LEFT-(PF4)
			RIGHT-(PF6)
			MID-(PE10)
	  按键：
			SET-(PE12)
			RST-(PE14)
	5,电位器：(PA3)
